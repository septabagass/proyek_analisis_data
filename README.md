# proyek_analisis_data
Proyek ini bertujuan untuk menganalisis pola penggunaan layanan bike sharing berdasarkan faktor waktu, musim, dan kondisi cuaca. Analisis dilakukan menggunakan pendekatan Exploratory Data Analysis (EDA) dan divisualisasikan melalui dashboard interaktif menggunakan Streamlit.

# Tujuan Proyek
- Mengidentifikasi pola penyewaan sepeda berdasarkan waktu (jam & hari)
- Menganalisis pengaruh cuaca terhadap jumlah penyewaan
- Membandingkan perilaku pengguna casual dan registered
- Menyajikan hasil analisis dalam bentuk dashboard interaktif

# Insight Utama
- Penyewaan meningkat signifikan pada jam sibuk (pagi & sore) → indikasi commuting
- Cuaca buruk menyebabkan penurunan drastis jumlah penyewaan
- Pengguna casual meningkat pada akhir pekan dan musim panas → aktivitas rekreasi
- Pengguna registered mendominasi penggunaan harian

# Tools & Teknologi
- Python
- Pandas
- Matplotlib & Seaborn
- Streamlit
